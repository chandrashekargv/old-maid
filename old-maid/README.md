# Old Maid Online Game

This project will be a full-stack online multiplayer Old Maid card game. It will use React for the frontend and Node.js (Express + WebSocket) for the backend to enable real-time play with friends.

## Features
- 2-8 players
- 52 cards + 1 joker
- Card shuffling and dealing
- Pair discarding
- Turn-based card picking
- Private hands
- Win/lose logic (including reverse mode)

## Setup
1. Backend: Express + WebSocket server
2. Frontend: React app
